Testing the basic functionalities:

webdriver methods:
------------------
1: WebDriver: WebDriver is used to open the required browser like firefox, chrome, internet explorer etc..,
syntax: WebDriver f= new FirefoxDriver();

2: get(); : get() is used to open the url like gmail.com , facebook.com etc..,
syntax: f.get("http://www.gmail.com");

3: findElement: findElement is used to access the text fields in an webpage that we want to open.
syntax:  f.findElement(By.Id("email")).sendKeys("bitra9999@gmail.com");

4: close(); : close() is used to close the browser that we opened by WebDriver() .
syntax: f.close();

5: By.Id: Entering the property id's for the required text fields.
6: click(); : Used to click the fields or buttons on a webpage
7: sendKeys(); : Used to pass the values for the text fields.
8: Navigate.to();
9: Navigate.forward();
10: Navigate.back(); All these navigate methods are used to navigate the page from one page to another page depending on the method that we used.
		    for example: from browser to url page by using the Navigate.to();
