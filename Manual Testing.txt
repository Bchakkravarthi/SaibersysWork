Creating the OBJECT and accessing the methods of other class:
------------------------------------------------------------------------------------
procedure:
--------------
first we created a class Test3 and declared a function Display() and we are going to display the content in the Display() function by creating an object of Test3 class in Test4 class.
	
	package test;

	public class Test3
	{
		void Display()
		{
			System.out.print("hi how are sir");
		}
	}



	package test;

	public class Test4 
	{

		public static void main(String[] args) 
		{
			Test3 t=new Test3();
			t.Display();
		
		}

	}
