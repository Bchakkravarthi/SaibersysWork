Basic Syntaxes in VB:
-------------------------

Variables:
	
	1: The Keyword used to declare to variables is "Dim".

	2: By Default the variables will be read as "String" .
	
	3: General systanxes for converting the variables to required Formats

		Cint : Converting any variants to "Integers type".
		Clng : Converting any variants to "Long type".
		Cstr: Converting any variants to "String type".
		Cdbl: Converting any variants to "Double type".
		Csng: Converting any variants to "Single Type".

	4: Keyword to print the output is " Msgbox".

	5: Keyword to enter the input at console is "Inputbox".
	
Basic_Programs_TypeConversations:
-------------------------------------------

Sample:
---------

	Dim a,b,c
	a=10
	b=20						//  Adding two numbers but we need to convert the numbers into integers.
	c=Cint(a)+Cint(b)

Control_Statements:
------------------------

	There will be Generally 3 types of Control statements, they are
		1: Conditional Statements
			
			IF , IF ELSE and ELSE IF Ladder

		2: Unconditional Statements
 
		3: Loop Statements

			FOR , WHILE , DO WHILE , SWITCH 

Sample:
---------

	Dim a
	a=Inputbox(" Enter the value of a ")
	If a=1 Then
	Msgbox(" SELENIUM QA TESTING ")
	ElseIf a=2 Then
	Msgbox(" SAIBERSYS ")      				// Using Conditional Control Statements 
	ElseIf a=3 Then
	Msgbox(" MAHESH : TUTOR ")
	ElseIf a=4 Then
	Msgbox(" ASHOK C BITRA")
	
	End If

Sample:

Synchronization:
--------------------

	Synchronization is mainly used to Delay the Execution Speed of our actions that we perform on the forms.

	Syntax:
		wait(seconds) // wait(5)
